## 一. 安装和配置Zshell
1. 安装

    sudo yum install zsh
2. 配置
自动化安装
    curl -L http://install.ohmyz.sh | sh

手动安装
    
    a. git clone git://github.com/robbyrussell/oh-my-zsh.git ~/.oh-my-zsh
    b. cp ~/.zshrc ~/.zshrc.orig
    c. cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc
    d. chsh -s /bin/zsh
    e. restart zsh
